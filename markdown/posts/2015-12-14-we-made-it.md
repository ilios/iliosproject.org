We are very pleased to announce that Ilios version 3.0.0 is now available for download.

Please use [this download link](https://github.com/ilios/ilios/releases/tag/v3.0.0) for download links, and to see the full release notes.

Live API documentation is available here: [https://ilios3-demo.ucsf.edu/api/doc](https://ilios3-demo.ucsf.edu/api/doc).

For additional information, including technical documentation, FAQs, and more, please visit the [Ilios wiki](https://github.com/ilios/ilios/wiki) at [https://github.com/ilios/ilios/wiki](https://github.com/ilios/ilios/wiki)

If you have any questions regarding installation or upgrade for this first of the 3rd generation Ilios code, please contact us at [support@iliosproject.org](mailto:support@iliosproject.org).
